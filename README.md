# Modbus Protocol Project - Vehicle Access Control System

## What is Modbus Protocol
1979년, Modicon에서 만든 시리얼 통신 프로토콜<br>
프로토콜이 단순하고 무료라서 산업 현장에서 전자 장치들을 서로 연결하는 목적으로 널리 사용된다.<br>
